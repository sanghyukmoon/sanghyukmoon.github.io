Using official python distribution with pip

Build from the source

  • Download latest source release for BOTH python2 and python3 from https://www.python.org/downloads/.
  • Configure: $ ./configure --enable-optimizations --with-lto --prefix=/usr/local
  • Build: $ make
  • Install: first, you should choose which version (2 or 3) is your "primary" version.
    Type $ sudo make install in your primary version directory.
    Type $ sudo make altinstall in your secondary version directory.
    (see Installing multiple versions in README)

Create virtual environment

venv (or virtualenv)
  • For python3, see https://docs.python.org/3.6/installing/index.html?highlight=virtual%20environment
  • For python2, see https://docs.python.org/2.7/installing/index.html?highlight=virtual%20environment
  • python3 -m venv ~/py3
  • virtualenv --python=python2.7 ~/py2

For easy use...

in bashrc...
  • alias py3='deactivate; source ~/py3/bin/activate'
  • alias py2='deactivate; source ~/py2/bin/activate'
  • source ~/py3/bin/activate
