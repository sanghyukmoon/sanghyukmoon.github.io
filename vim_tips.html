VIM tips

Syntax highlighting for files with odd extensions

Say you have a working code main.cpp and a backup code main.cpp.bck. You display them in a single window by :vsp. You will notice the text for the file main.cpp.bck is not highlighted properly because the file name does not end with .cpp. In that case, simply type :set syntax=cpp.

Enable mouse for a terminal VIM

Do you use gvim just because it supports mouse? you can use mouse also in vim! Just type :set mouse=a (see :help mouse). You can drag the vertical split line to adjust the window size. You can do visual selection using your mouse.

Execute python script inside VIM by pressing <F9>

autocmd FileType python nnoremap <buffer> <F9> :exec '!python' shellescape(@%, 1)
