우분투 ALT R 키로 한영전환

  • ibus 대신 fcitx 설치한다.
  • sudo apt install fcitx-hangul
  • System Settings - Language Support - Keyboard input method system = fcitx
  • System Settings - Keyboard - Shortcuts - Typing: 모두 disabled로 전환 (Backspace 입력)
  • Compose key = Right Alt
  • Switch to next source = Multikey (press Alt R)
