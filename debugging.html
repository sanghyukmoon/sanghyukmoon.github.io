Debugging tips

AddressSanitizer

  • Compile with -fsanitize=address to detect memory errors such as out-of-bound access
  • For complete description, see https://github.com/google/sanitizers/wiki/AddressSanitizer
  • Compile a specific file seperately with -fsanitize=address (not in the Makefile) to investigate that file alone. You have to relink with -fsanitize=address

Floating point exception

#include <fenv.h>
feenableexcept(FE_DIVBYZERO | FE_INVALID | FE_OVERFLOW);
  • For complete description, see https://linux.die.net/man/3/feenableexcept
